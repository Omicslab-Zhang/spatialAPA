# spatialAPA
Source code of spatialAPAdb (http://www.biomedical-web.com/spatialAPAdb/home) data processing.

## An integrated pipeline enable gene expression, APA/PSI, weight 3'UTR length for 10x visium data
SCAPE (https://github.com/LuChenLab/SCAPE) should be pre-installed before using this pipeline.
